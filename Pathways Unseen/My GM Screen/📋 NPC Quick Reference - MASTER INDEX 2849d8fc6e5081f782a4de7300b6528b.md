## Threat Levels (CR)

**BOSS TIER**

- Vorgoth (CR 15) - Has war-mechs, Iron Order backup

**DON TIER**

- Thorgar (CR 10) - Never retreats, honorable tank
- Azrael (CR 9) - Escape artist, uses minions
- Grommash (CR 8) - Berserker, reckless aggression
- Kethar (CR 8) - Tactical modern, Action Surge danger
- Iron Lord (CR 8) - Mystery, summons The Silent

**LIEUTENANT TIER**

- Kethris (CR 9) - Only if contracted, 100% lethal
- Iron Order Captain (CR 8) - Respawns in 24hrs
- Jevan (CR 7) - Avoids combat, talks out
- Veyran (CR 7) - Shadow monk, protects Iblith

**WAND TARGETS**

- Silra (CR 6) - HARD - Etherealness escape
- Zethrin (CR 5) - MEDIUM - Coward, surrenders
- Krev (CR 4) - EASY - Doesn't value wand

## Combat Priority

**If Fighting Multiple Dons:**

1. Azrael first (flees early)
2. Control Grommash (Reckless = easy hits)
3. Tank Thorgar (won't flee)
4. Avoid Kethar early (6 attacks)
5. Never solo Iron Lord (summons help)

**If Fighting Authority:**

- Vorgoth = Run or TPK
- Iron Order = Strong but non-lethal

**If Fighting Bregan:**

- Jevan talks, doesn't fight
- Veyran extracts Iblith
- Kethris = RUN if contracted

## Negotiation Leverage

**What Each Wants:**

- Iron Hammer: Respect, honor, tradition
- Deepstone: Fair wages, dock control
- Brotherhood Forge: Rare materials, eliminate rivals
- Iron Mine: Debt payments, expansion
- Steel Syndicate: Territory, modernization
- Bregan: Information, profitable chaos
- Vorgoth: Stability, balance

**What Each Fears:**

- Iron Hammer: Being forgotten
- Deepstone: Worker uprising
- Brotherhood Forge: Public exposure
- Iron Mine: Identity revealed
- Steel Syndicate: Vorgoth's mechs
- Bregan: Total collapse
- Vorgoth: One family dominating
